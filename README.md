# Hiragana-Identifier
Machine Learning project to identify Japanese characters (hiragana) from a dataset. 

Dataset available at: [Hiragana73](https://lab.ndl.go.jp/cms/hiragana73?fbclid=IwAR2isHvlc2sxjzytRbHRXDaNQM__sevaA9azydGpcrUqlgXdK8LcMpXi13E) and 
[HiraganaGit](https://github.com/inoueMashuu/hiragana-dataset)
